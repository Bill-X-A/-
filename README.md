requirements.txt 是什么
你的电脑上已经装好了所有需要的库（zhipuai、streamlit等），但 Streamlit Cloud 服务器是一台全新的电脑，什么都没装。
requirements.txt 就是一张”购物清单”，告诉服务器：
“运行我的程序需要这些库，请帮我装上”
类比
你在自己家做饭（本地电脑），冰箱里有齐全的食材。
但朋友要在他家（Streamlit服务器）做同样的菜，你得给他一张食材清单，他才知道要买什么。
怎么知道要写哪些
规律很简单：你代码里 import 了什么库，就在 requirements.txt 里写什么。
from zhipuai import ZhipuAI       → requirements.txt 写 zhipuai
from sentence_transformers import...  → requirements.txt 写 sentence-transformers
import faiss                       → requirements.txt 写 faiss-cpu​​​​​​​​​​
